Permitir imprimir receta desde la aplicación.

### Reportes

Reportes sugeridos:

- Pacientes atendidos por fecha
- Medicamentos entregados
- Stock bajo
- Historial por paciente

### Exportación

Exportar datos a:

- Excel
- CSV
- PDF

### Usuarios

Crear módulo de usuarios:

- Administrador
- Doctor
- Auxiliar

### Roles y permisos

Permisos por módulo:

- Ver
- Crear
- Editar
- Eliminar

### Backup

Crear backup automático de MySQL.

### Electron Builder

Empaquetar la aplicación como `.exe`.

### Mejoras UI

- Tema oscuro
- Iconos
- Animaciones
- Gráficos en dashboard
